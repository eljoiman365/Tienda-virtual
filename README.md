## 👜 Tienda-virtual 👜

Este código desarrollado en JAVA, consiste en simular una tienda virtual, con funcionalidades como el historial de compra, carrito de compras, catalogo de productos, entre otros...

## 🧰 Herramientas utilizadas:

🟢 Java

🟢 Framework Spring

🟢 IntelliJ IDEA
